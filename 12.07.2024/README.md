# Симулятор овощного магазина
## тех. задание
### моделирование бизнес процессов овощного магазина
- реализовать возможность покупки продажи товара
- товар делится на категории: овощи и фрукты
- у товара будет цена, название, категория
- при покупке будет формироваться чек
- в чеке указано позиция, наименование, вес, 
цена на шт/кг, общая цена позиции, общая цена чека, 
уникальный идентификатор чека
- при покупке каждый чек будет записываться в файл 
в папке cheks а наименование файла будет соответствовать его id
